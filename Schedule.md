## 1 用户相关

### 1.1 登录与注册

接口信息如下：

|  key   | value  |
|  ----  | ----  |
| 访问地址  | `/user/login` |
| 访问方式  | post |       

post所需要的参数：

| 参数名 | 参数值（样例）| 描述 | 可否为空 |
| ----   | ----- | ---- | ---- |
| account | 18651537538 | 表示用户的账号 | 否 |
| passwd | 123456 | 表示用户的密码 | 否 |

应返回的数据：

| 参数名 | 参数值（样例）| 描述 |
| ---- | ---- | ---- |
| account | 18651537566 | 描述用户的账号 |
| username | indoor_7566 | 描述用户的昵称 |
| token | wxd930ea5d5a258f4f | 描述身份令牌 |
| timesampe | 119775345 | 描述接口返回数据时的时间戳 |
